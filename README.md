#Parking-Lot
##requirements:
```
scala : 2.12
java  : 1.8
sbt   : 1.2.8
```

### Used Language : Scala
Require proper scala and sbt set up for project

##Usage:

####build : 
`./bin/setup`
or
`sbt clean compile test assembly` (from the project root which is `.` here)

####run :
#####file mode:
```
./bin/parking_lot file_path
```
#####interactive mode:
```
./bin/parking_lot
```
